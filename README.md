# 豪士藜麦吐司 - 梦幻烘焙工坊

大广赛参赛作品 - 微信小程序互动游戏

## 作品简介

这是一款以豪士藜麦吐司为主题的微信小程序互动游戏。通过四个精心设计的关卡，让用户在娱乐中了解豪士品牌的核心卖点：全球精选原料、透明工厂工艺、无边吐司特色。

## 游戏关卡

1. **原料引力场** - 移动托盘接住全球精选食材
2. **去边魔术秀** - 滑动切掉吐司边，体验无边工艺
3. **烘焙流水线** - 完成6道生产工序（和面、压面、切割、烘焙、冷却、包装）
4. **分享工厂狂欢** - 展示成果，领取优惠券，分享给好友

## 技术栈

- Taro 4.x
- React 18
- TypeScript
- Tailwind CSS 4.x
- Zustand

## 快速开始

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm run dev

# 构建微信小程序
pnpm run build:weapp
```
